## Project Overview
This project demonstrates basic data analysis and visualization using Python.

### Part 1: Automated Summary (NumPy)
- Calculates mean and standard deviation
- Shows effect of adding new data

### Part 2: Data Visualization (Matplotlib)
- Scatter plots
- Prediction lines
- Comparison of multiple weights

## Libraries Used
- NumPy
- Matplotlib
